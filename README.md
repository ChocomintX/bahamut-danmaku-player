# 动画疯弹幕播放器

一个桌面应用，用来抓取巴哈姆特动画疯的弹幕，并把弹幕和你的本地视频文件同步播放。

它不会下载动画疯的视频流，只会读取公开可访问的弹幕与番剧目录信息，方便你把本地视频和动画疯弹幕对上。

## 现在能做什么

- 搜索动画疯番剧，并读取可选剧集列表
- 或者直接输入 `videoSn` / 动画疯播放页链接加载弹幕
- 默认同时抓取 `TW` 和 `HK` 两个地区源
- 一次拖入多集本地视频，并按文件名自动尝试匹配剧集
- 支持全屏弹幕、点击画面播放/暂停、双击全屏
- 支持长按方向键 2 倍快进 / 倒退
- 支持弹幕时间偏移、透明度、字号、速度、位置开关
- 支持繁体弹幕转简体
- 支持导出弹幕 JSON

## 适用场景

- 你已经有本地视频文件
- 想继续看动画疯的弹幕
- 希望多集视频能一次导入、快速切换

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建 Windows 安装包：

```bash
npm run package:win
```

## 使用流程

1. 先导入一个或多个本地视频文件
2. 在左侧搜索番剧，或者直接输入 `videoSn`
3. 选中剧集后加载弹幕
4. 在右侧播放器中开始播放，并按需要调整弹幕参数

## 已知限制

- 这是偏实用型的桌面工具，不是动画疯官方客户端
- 弹幕轨道避让是近似实现，不是逐帧复刻原播放器
- 剧集自动匹配主要依赖文件名里的集数信息
- 目前 release 主要提供 Windows 安装包
